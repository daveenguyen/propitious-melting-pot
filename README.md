# Propitious Melting Pot
This repo is a result of my playing around with many things.
 - [Docker][docker]
 - [Docker Compose][docker-compose]
 - [Nginx][nginx]
 - [Ghost][ghost]
 - [StartSSL][start-ssl]

## Running
Install [Docker Compose][docker-compose]
Inside this folder, run `docker-compose up -d`
